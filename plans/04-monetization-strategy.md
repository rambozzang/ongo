## 1. Executive Summary

onGo의 현재 4단계 가격 체계(Free/Starter/Pro/Business)는 한국 크리에이터 시장의 특성과 경쟁 환경을 충분히 반영하지 못하고 있다. 본 기획서는 Publer 대비 명확한 가격 우위를 확보하면서도 지속 가능한 수익 모델을 구축하기 위한 전략을 제시한다.

**핵심 전략 요약:**
- 4단계를 3단계 + Enterprise로 재편 (가격 단순화)
- Free 플랜의 전략적 확대로 사용자 풀 확보 후 AI 크레딧 기반 수익화
- Publer $12/월(약 16,000원) 대비 9,900원에 동등 이상 기능 제공
- 에이전시/MCN 전용 B2B 가격으로 대량 계정 수익 확보
- 한국 간편결제 완전 통합으로 결제 마찰 최소화

**목표 지표 (출시 후 12개월):**

| 지표 | 목표 |
|------|------|
| 무료 가입자 | 50,000명 |
| 유료 전환율 | 8-12% (한국 SaaS 평균 5% 대비) |
| 월 ARPU (유료 사용자) | 15,000원 |
| AI 크레딧 추가 구매율 | 유료 사용자의 35% |
| 월간 이탈률 (Churn) | 5% 이하 |
| 에이전시 계정 비중 | 매출의 25% |

## 2. 현재 가격 체계 진단

### 2.1 현행 구조 (코드 기반 분석)

현재 `PlanType.kt`에 정의된 플랜 구조:

| 항목 | Free | Starter (9,900원) | Pro (19,900원) | Business (49,900원) |
|------|------|-----|-----|---------|
| 연동 플랫폼 | 1개 | 3개 | 4개 | 4개 |
| 월 업로드 | 5회 | 30회 | 100회 | 무제한 |
| 예약 업로드 | 불가 | 7일 | 30일 | 90일 |
| 분석 기간 | 7일 | 30일 | 365일 | 무제한 |
| 저장 공간 | 1GB | 10GB | 50GB | 200GB |
| 무료 AI 크레딧 | 30 | 100 | 300 | 1,000 |
| 팀원 | 0명 | 0명 | 2명 | 10명 |

### 2.2 문제점 분석

#### 문제 1: Free 플랜이 지나치게 제한적

- **플랫폼 1개 연동**: onGo의 핵심 가치인 "멀티플랫폼 동시 업로드"를 Free에서 경험할 수 없음
- **월 5회 업로드**: 크리에이터가 제품 가치를 느끼기 전에 한계에 도달
- **Publer 비교**: Publer Free도 3 소셜 계정 허용 (최근 5에서 축소했지만 여전히 3개)
- **영향**: Free 사용자가 가치를 체감하지 못해 유료 전환 동기 부족

#### 문제 2: Starter(9,900원)와 Pro(19,900원) 간 가치 격차 불균형

- Starter에서 Pro로 가면 가격이 2배인데, 플랫폼은 3개에서 4개로 1개만 증가
- 핵심 차별점(팀원 기능)이 Pro부터인데 개인 크리에이터에게는 불필요
- 분석 기간(30일 vs 365일)은 큰 차이이나, 이것만으로 2배 가격을 정당화하기 어려움

#### 문제 3: Business(49,900원) 가격 장벽

- Pro(19,900원) 대비 2.5배 점프
- 대상 고객(에이전시/MCN)에게는 좌석/계정 수가 핵심인데 팀원 10명 한계
- B2B 대량 고객을 위한 유연한 가격 체계 부재

#### 문제 4: AI 크레딧과 구독 간 연결 부재

- 현재 크레딧 패키지(`CreditPackage.kt`)가 구독 플랜과 가격이 동일하게 설정됨 (4,900/9,900/19,900/49,900원)
- 사용자가 구독 업그레이드 대신 크레딧만 구매하는 "크레딧 유형의 카니발리제이션" 가능성
- 크레딧 유효기간(30/60/90/180일)이 짧아 구매 심리 저해

#### 문제 5: 연간 결제 할인 UI 미구현

- `BillingCycle` enum에 `MONTHLY`와 `YEARLY`가 정의되어 있지만, 프론트엔드 `SubscriptionView.vue`에서 연간 결제 토글이 없음
- 연간 결제 할인은 이탈률을 낮추는 가장 효과적인 방법 중 하나

## 3. 가격 체계 재설계

### 3.1 새 플랜 구조: 3+1 체계

기존 4단계를 **3단계 개인 플랜 + 에이전시 플랜**으로 재편한다.

```
Free (0원) --> Creator (7,900원/월) --> Creator Pro (15,900원/월) --> Agency (별도 협의)
```

| 항목 | Free | Creator | Creator Pro | Agency |
|------|------|---------|-------------|--------|
| **월 가격** | 0원 | 7,900원 | 15,900원 | 49,900원~ |
| **연간 가격 (월환산)** | - | 5,900원/월 (70,800원/년) | 11,900원/월 (142,800원/년) | 별도 협의 |
| **연동 플랫폼** | **2개** | **4개** (전체) | **4개** (전체) | **무제한** (하위 계정 포함) |
| **월 업로드** | **10회** | **50회** | **무제한** | **무제한** |
| **예약 업로드** | 3일 | 14일 | 60일 | 90일 |
| **분석 기간** | 7일 | 90일 | **무제한** | **무제한** |
| **저장 공간** | 2GB | 20GB | 100GB | 500GB~ |
| **무료 AI 크레딧/월** | **50** | **200** | **500** | **2,000** |
| **팀원** | 0명 | 1명 | 5명 | 무제한 |
| **댓글 관리** | 읽기만 | 전체 | 전체 + AI 답변 | 전체 + AI 답변 |
| **고객 지원** | 커뮤니티 | 이메일 (24시간) | 우선 이메일 + 채팅 | 전담 매니저 + Slack |

### 3.2 변경 근거

#### Free 플랜 확대 (1개 플랫폼 -> 2개, 5회 -> 10회, 크레딧 30 -> 50)

**이유**: 한국 SaaS 시장에서 무료 사용자가 가치를 체감해야 전환한다. "2개 플랫폼 동시 업로드"를 경험한 사용자가 "4개 플랫폼"으로 확장하고 싶은 욕구가 전환 동기가 된다.

**벤치마크:**
- Publer Free: 3 소셜 계정 (유저 불만으로 5에서 축소했지만 여전히 3개)
- Buffer Free: 3 채널
- Hootsuite Free: 2 소셜 계정 (2023년 무료 폐지)
- Notion Free: 무제한 개인 사용

**예상 효과:**
- Free 가입자 수: 30% 증가 예상 (현 대비)
- Free -> 유료 전환율: 5% -> 10% 예상 (가치 체감 증가)

#### Starter -> Creator (9,900원 -> 7,900원)

**이유**: 한국 크리에이터의 월 수익 중앙값이 50만원 미만이라는 점을 고려하면, 9,900원은 심리적 저항선이 높다. 7,900원은 "만원 미만"이라는 심리적 이점이 있으며, Publer $12(약 16,000원) 대비 절반 가격이라는 강력한 포지셔닝이 가능하다.

**가격 심리학 근거:**
- 9,900원: "거의 만원" -> 저항감
- 7,900원: "팔천원도 안 됨" -> 수용성 증가
- 한국 모바일 구독 서비스 평균: 7,000-9,000원대 (넷플릭스 베이직 5,500원, 유튜브 프리미엄 10,450원)

**수익 시뮬레이션:**
- 가격 인하(9,900 -> 7,900원, -20%)로 전환율 40% 증가 시: 순 매출 12% 증가
- 가격 탄력성 계수 -2.0 적용 (한국 SaaS 시장 추정치)

#### Pro -> Creator Pro (19,900원 -> 15,900원)

**이유**: "만오천원대"는 한국 소비자에게 "합리적 프리미엄" 영역이다. 19,900원과의 4,000원 차이가 업그레이드 결정을 크게 좌우한다.

**추가 가치:**
- 무제한 업로드(기존 100회 제한 해제)
- 분석 기간 무제한
- AI 크레딧 300 -> 500 (67% 증가)
- 팀원 2명 -> 5명

## 4. 프리미엄 전환 전략

### 4.1 전략 체계도

```
[가입] -> [온보딩 체험] -> [가치 실현 (Aha Moment)] -> [한계 도달] -> [전환 유도] -> [유료 결제]
     14일 프로 체험      2개 플랫폼 동시 업로드       업로드/크레딧 소진      스마트 넛지
```

### 4.2 14일 Creator Pro 트라이얼

**설계:**
- 가입 즉시 14일간 Creator Pro 전 기능 체험
- 결제 정보 입력 불요 (한국 시장에서 사전 카드 등록은 전환율을 크게 낮춤)
- 트라이얼 종료 시 Free로 자동 전환 (강제 결제 없음)
- 트라이얼 중 업로드한 콘텐츠와 분석 데이터는 보존

**트라이얼 중 제공하는 것:**
- 4개 플랫폼 연동
- 무제한 업로드
- AI 크레딧 100개 보너스 (트라이얼 전용, 이월 불가)
- 전체 분석 기능

**벤치마크 데이터:**
- Canva Pro 30일 무료 체험: 전환율 약 15%
- Notion Plus 7일 체험: 전환율 약 8%
- Buffer 14일 체험: 전환율 약 10%
- **목표: 14일 체험 후 전환율 12%**

**구현 요구사항:**
- `Subscription` 도메인에 `trialEndDate: LocalDateTime?` 필드 추가
- `SubscriptionStatus`에 `TRIAL` 상태 추가
- 트라이얼 만료 스케줄러 (`BillingScheduler`에 추가)
- 트라이얼 7일차, 3일차, 1일차에 자동 알림 발송

### 4.3 기능 잠금 + 미리보기 (Soft Lock)

완전히 차단하는 대신, "미리보기"를 보여주고 잠금 해제를 유도한다.

| 기능 | Free에서의 경험 | 전환 유도 방식 |
|------|----------------|---------------|
| AI 메타데이터 생성 | 첫 3회 무료 체험 후 크레딧 필요 | "이 기능으로 생성한 제목의 CTR이 평균 23% 높습니다" |
| 분석 대시보드 | 7일 데이터만 표시, 30일 그래프는 블러 처리 | "30일 트렌드를 보면 최적 업로드 시간을 알 수 있습니다" |
| 예약 업로드 | 3일 후까지만 가능, 달력에 14일 이후 날짜 회색 표시 | "주간 업로드 스케줄링으로 일관된 채널 성장을 달성하세요" |
| 멀티플랫폼 | 2개까지, 3/4번째 플랫폼 슬롯 잠금 표시 | "유튜브 + 틱톡에 추가로 인스타 릴스와 네이버 클립까지!" |

### 4.4 AI 크레딧 맛보기 전략

AI가 onGo의 핵심 차별점이므로, AI 기능 체험이 전환의 트리거가 되어야 한다.

**첫 업로드 AI 자동 체험:**
1. 사용자가 첫 영상을 업로드하면 자동으로 AI 메타데이터 생성 실행 (크레딧 차감 없음)
2. "AI가 생성한 제목/설명이 이렇게 최적화됩니다" 결과를 보여줌
3. 두 번째 업로드부터 크레딧 소진
4. 크레딧 소진 시 "크레딧 충전" 또는 "Creator 플랜 업그레이드" 양갈래 CTA

**크레딧 소진 시 전환 유도 UX:**

```
[크레딧이 부족합니다]

현재 남은 크레딧: 3 / 50 (월 무료)

옵션 1: Creator 플랜 업그레이드 (7,900원/월)
  -> 매월 200 크레딧 + 4개 플랫폼 + 50회 업로드
  -> "크레딧 200개를 따로 사면 약 3,900원인데,
      7,900원에 모든 프리미엄 기능까지!"

옵션 2: 크레딧만 충전
  -> 스타터 팩 300크레딧 2,900원
```

**예상 전환 경로 비율:**
- 구독 업그레이드: 60%
- 크레딧만 구매: 25%
- 이탈: 15%

### 4.5 전환 넛지 타이밍

| 트리거 | 넛지 내용 | 채널 |
|--------|----------|------|
| 업로드 한도 80% 도달 | "이번 달 업로드 8/10회 사용. Creator 플랜으로 50회까지!" | 인앱 배너 |
| AI 크레딧 잔여 10 이하 | "AI 크레딧이 부족해지고 있어요. 지금 충전하면 10% 보너스!" | 인앱 모달 + 푸시 |
| 트라이얼 종료 3일 전 | "Pro 기능이 3일 후 종료됩니다. 지금 구독하면 첫 달 30% 할인!" | 이메일 + 인앱 |
| 첫 동시 업로드 성공 | "축하합니다! 2개 플랫폼에 동시 업로드 완료. 4개 플랫폼은 어떨까요?" | 인앱 토스트 |
| 분석에서 성장 감지 | "지난주 대비 조회수 150% 증가! 더 깊은 분석으로 성장을 가속화하세요" | 이메일 |
| 비활성 7일 경과 | "놓치고 계신 인사이트가 있어요. AI가 최적 업로드 시간을 분석했습니다" | 이메일 + 푸시 |

## 5. AI 크레딧 경제

### 5.1 AI 기능별 크레딧 소비 재설계

현재 `AiFeature.kt` 기반 분석 및 개선안:

### 5.3 크레딧 패키지 재설계

현재 크레딧 패키지의 문제: 구독 플랜과 가격대가 겹쳐 카니발리제이션 발생.

**새 크레딧 패키지:**

| 패키지 | 크레딧 | 가격 | 크레딧당 단가 | 유효기간 | 할인율 (대비 기본 단가) |
|--------|-------|------|-------------|---------|----------------------|
| 라이트 | 100 | 1,900원 | 19.0원 | 30일 | 기본 |
| 스탠다드 | 300 | 4,900원 | 16.3원 | 60일 | 14% 할인 |
| 프리미엄 | 800 | 9,900원 | 12.4원 | 90일 | 35% 할인 |
| 메가 | 2,000 | 19,900원 | 10.0원 | 120일 | 47% 할인 |
| 엔터프라이즈 | 5,000 | 39,900원 | 8.0원 | 180일 | 58% 할인 |

**변경 포인트:**
- 가격대를 구독과 분리 (1,900/4,900/9,900/19,900/39,900)
- 최소 구매 단위를 낮춤 (500 -> 100): 부담 없는 "맛보기" 가능
- 대량 구매 시 단가 할인 폭 확대: 자연스럽게 큰 패키지로 유도
- 유효기간 연장: 30일 -> 120일/180일로 구매 심리 장벽 해소

**크레딧당 단가 비교 (구독 포함 무료 크레딧 vs 별도 구매):**
- Creator 플랜 (200 크레딧/7,900원): 크레딧당 39.5원 -- 하지만 다른 기능 포함
- 스탠다드 팩 (300 크레딧/4,900원): 크레딧당 16.3원
- 결론: 구독은 "기능 + 크레딧 번들", 크레딧 팩은 "순수 AI 추가 사용"으로 포지셔닝

### 5.4 크레딧 소진 유도 메커니즘

AI 크레딧의 수익화를 극대화하기 위한 "자연스러운 소진" 전략:

1. **자동 추천 팝업**: 업로드 완료 시 "AI로 제목 최적화할까요?" 자동 제안
2. **일괄 처리 기능**: "이번 달 업로드 8개 전체에 해시태그 한번에 추천받기" (크레딧 16개 소진)
3. **주간 리포트 자동 생성**: Creator Pro 이상에서 주간 AI 성과 리포트 자동 생성 (옵트아웃 방식)
4. **크레딧 만료 임박 알림**: "스탠다드 팩 잔여 45크레딧이 7일 후 만료됩니다. 지금 사용하세요!" -> 소진 후 재구매 유도

**연간 결제 유도 전략:**
- 기본 표시 가격을 연간 기준으로 ("월 5,900원~"), 월간 가격은 부가 정보
- 첫 연간 결제 전환 시 추가 1개월 무료 (총 13개월)
- 연간 결제 시 AI 크레딧 2개월치 보너스 지급

**예상 효과:**
- 연간 결제 비율: 전체 유료 사용자의 40% (목표)
- 연간 결제 사용자 이탈률: 월간 대비 60% 감소 (산업 평균)

### 8.2 로열티 크레딧 프로그램

**연속 구독 보상:**

| 구독 기간 | 보너스 크레딧 (매월 추가) | 누적 혜택 |
|----------|------------------------|----------|
| 3개월 연속 | +20 크레딧/월 | "감사 보너스" |
| 6개월 연속 | +50 크레딧/월 | "실버 크리에이터" 배지 |
| 12개월 연속 | +100 크레딧/월 | "골드 크리에이터" 배지 + 연간 성과 리포트 무료 |
| 24개월 연속 | +200 크레딧/월 | "플래티넘 크리에이터" 배지 + 전용 기능 얼리 액세스 |

**이탈 시 보상 손실**: 구독을 해지하면 연속 구독 보상이 리셋되므로, 장기 구독자일수록 이탈 비용이 증가한다.

### 8.3 이탈 방지 플로우 (Cancellation Flow)

현재 `SubscriptionView.vue`에서 "취소" 버튼을 누르면 단순 확인 모달만 표시된다. 이를 다단계 이탈 방지 플로우로 개선한다.

**Step 1: 이탈 이유 조사**
```
왜 떠나시나요?
[ ] 가격이 비싸요
[ ] 원하는 기능이 없어요
[ ] 다른 서비스를 사용해요
[ ] 크리에이터 활동을 중단해요
[ ] 기타: ___________
```

**Step 2: 이유별 맞춤 제안**

| 이유 | 제안 |
|------|------|
| 가격이 비싸요 | "다음 3개월 50% 할인 적용해 드릴까요?" (Creator: 3,950원/월) |
| 기능이 없어요 | "어떤 기능이 필요하세요? 피드백 주시면 우선 개발 검토합니다 + 1개월 무료 연장" |
| 다른 서비스 | "어떤 서비스인지 알려주시면 비교해 드릴게요 + 1개월 무료 연장" |
| 활동 중단 | "구독을 최대 3개월 일시정지할 수 있습니다 (데이터 보존)" |

**Step 3: 최종 제안 (Step 2 거절 시)**
```
정말 떠나시겠어요?

지금 해지하시면:
- 연속 구독 보상 (6개월차, +50 크레딧/월)이 초기화됩니다
- 현재 분석 데이터 90일 보관 후 삭제됩니다
- 업로드한 영상 메타데이터 30일 보관 후 삭제됩니다

[구독 유지하기]  [일시정지 (최대 3개월)]  [해지하기]
```

**예상 효과:**
- 이탈 방지 플로우 도입 시 해지 요청 대비 실제 해지율: 60% (40% 리텐션)
- 할인 제안 수락률: 25%
- 일시정지 선택률: 15%
- 벤치마크: Netflix 해지 플로우 유지율 30%, Spotify 35%

### 8.4 구독 일시정지 기능

**설계:**
- 최대 3개월 일시정지 가능 (연 1회)
- 일시정지 중: 결제 없음, Free 플랜 기능으로 전환, 데이터 보존
- 재개 시: 이전 플랜으로 자동 복구, 연속 구독 기간 유지
- 일시정지 만료 시: 자동 재개 또는 해지 선택

**구현:**
- `SubscriptionStatus`에 `PAUSED` 상태 추가
- `Subscription`에 `pausedAt`, `pauseUntil` 필드 추가
- `BillingScheduler`에 일시정지 만료 처리 로직 추가

### 8.5 데이터 락인 (Data Lock-in)

시간이 지날수록 onGo를 떠나기 어렵게 만드는 자연스러운 전환 비용:

1. **분석 히스토리 축적**: 6개월치 크로스 플랫폼 분석 데이터는 다른 도구에서 재현 불가
2. **AI 학습 데이터**: 사용할수록 AI 추천이 정확해진다는 메시지 (개인화)
3. **예약 업로드 캘린더**: 몇 주치 예약이 설정되어 있으면 전환 비용 증가
4. **팀 협업 데이터**: 에이전시 고객은 팀 워크플로우가 설정되면 전환이 매우 어려움

### 9.2 포지셔닝 전략: "한국 크리에이터 전용, 절반 가격에 두 배 기능"

**핵심 메시지:**

> "글로벌 툴의 절반 가격, 한국 크리에이터에게 필요한 건 두 배로."

**세부 포지셔닝:**

1. **가격 우위**: "Publer보다 50% 저렴한 7,900원"
   - 한국 크리에이터가 USD 결제 시 환율 + 해외결제 수수료 부담
   - 원화 결제 + 간편결제 = 체감 가격 차이 더 큼

2. **한국 특화 우위**: "네이버 클립, 유일한 지원"
   - Publer는 네이버 클립 미지원 (글로벌 서비스라 한국 로컬 플랫폼 무관심)
   - 한국 크리에이터에게 네이버 클립은 필수 (네이버 SEO + 추가 수익)

3. **AI 우위**: "한국어 네이티브 AI, 진짜 한국어를 알아요"
   - Publer AI는 영어 기반, 한국어는 번역 품질 수준
   - onGo AI는 한국 트렌드, 한국어 맞춤법, 한국 SEO 최적화

4. **영상 전문 우위**: "텍스트 포스팅 도구가 아닌, 영상 전문 도구"
   - Publer는 텍스트/이미지 포스팅이 메인, 영상은 부가
   - onGo는 영상 업로드/관리가 핵심 (STT, 스크립트 분석, 섬네일 최적화 등)

## 부록 C: 용어 정의

| 용어 | 정의 |
|------|------|
| MRR | Monthly Recurring Revenue - 월 반복 매출 |
| ARPU | Average Revenue Per User - 유저당 평균 매출 |
| Churn | 이탈률 - 일정 기간 내 해지한 유료 사용자 비율 |
| LTV | Lifetime Value - 고객 생애 가치 |
| CAC | Customer Acquisition Cost - 고객 획득 비용 |
| NRR | Net Revenue Retention - 순매출유지율 (기존 고객 매출 성장 포함) |
| Payback Period | CAC 회수 기간 - LTV/CAC 기준 |
| Gross Margin | 매출총이익률 - (매출-직접비용)/매출 |
